"Write test data" stamps localStorage + a cookie + a cache record with the current mode & time. Do it in Safari, then install and reopen here: anything still showing "written in: browser" crossed the boundary.

How to test

  1. Open this page in Safari (optionally add a query like ?grp=test123).
  2. Tap Write test data.
  3. Share → Add to Home Screen.
  4. Open the new icon. Check: did the Launch URL keep the query? Is localStorage / cookie / cache still there, and what mode were they "written in"?

# Adding a new group

Each meditation group gets its own page, app icon, and QR code. Creating one
takes about a minute and needs no technical knowledge — you just type the
group's name.

## Create a group

1. **Click this link:**
   👉 [**Create a new group**](https://github.com/bigjosh/AdamTimer/issues/new?labels=New%20Group&body=Add%20the%20new%20group%20name%20in%20the%20title%20field%20above%2C%20then%20press%20the%20create%20button%20below.%20The%20link%20and%20QR%20code%20will%20appear%20as%20a%20comment%20below%20in%20a%20few%20seconds.)
   (If asked, sign in to GitHub first.)
2. In the **title** field at the top, type **the group's name** — for example:
   `Homeboy Industries`. That's the only thing you need to fill in.
3. Click the green **Create** (or **Submit new issue**) button. (The description
   box already has a short reminder in it — you can leave it as-is.)
4. Within a minute a comment appears with the group's **link and QR code**, and
   the issue closes itself. Share that link or QR with the group — done.

> **Note:** the name is permanent (it's baked into the link and QR code), so
> check the spelling before submitting. If a name is ever rejected as a
> duplicate, just submit again with a slightly different name.

# Adding a new group

Each meditation group gets its own page, app icon, and QR code. Creating one
takes about a minute and needs no technical knowledge — you just type the
group's name.

## Create a group (the easy way)

1. **Click this link:**
   👉 [**Create a new group**](https://github.com/bigjosh/AdamTimer/issues/new?title=New%20group&labels=New%20Group)
   (If asked, sign in to GitHub first.)
2. The page opens with the **title** and the **New Group** label already filled
   in. **Leave the title exactly as it is** — don't change it.
3. In the big **"Add a description"** box, type **just the group's name** —
   for example: `Homeboy Industries`. Nothing else.
4. Click the green **Create** (or **Submit new issue**) button.
5. Within a minute a comment appears with the group's **link and QR code**, and
   the issue closes itself. Share that link or QR with the group — done.

> **Note:** the name is permanent (it's baked into the link and QR code), so
> check the spelling before submitting. If a name is ever rejected as a
> duplicate, just submit again with a slightly different name.

---

## One-time setup: the "New Group" label

The link above tags each request with a **New Group** label so they're easy to
find later. The label is created and maintained automatically, so you normally
don't need to do anything. If it's ever missing, here's how to recreate it by
hand:

1. Open the repository: **https://github.com/bigjosh/AdamTimer**
2. Click the **Issues** tab.
3. Click **Labels** (next to the search box).
4. Click the green **New label** button.
5. For **Label name**, type: `New Group`
6. Optionally add a description (e.g. *Creates a new AdamTimer group bundle*) and
   pick a color.
7. Click **Create label**.
